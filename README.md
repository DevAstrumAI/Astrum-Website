# Astrum AI Web App
A React-based AI product website with services, technologies, career, about and blog modules.

## Purpose
This project is built to showcase Astrum AI services, technologies, and portfolio
with a scalable and modular frontend architecture.

## Tech Stack
- React (Vite)
- Tailwind CSS
- JavaScript (ES6+)
- Node.js (v18+)

## Setup Instructions
1. Clone the repository
   git clone https://github.com/DevAstrumAI/Website.git

2. Install dependencies
   npm install

3. Start development server
   npm run dev

## Features
## Homepage

The homepage serves as the central introduction to the company. It provides a complete overview of the organization, including its vision, structure, services, and solutions. This page highlights the company’s overall definition, industry presence, service offerings, and client messages to establish credibility and trust.

## About

The About section focuses on the company’s identity and background. It includes the company overview, CEO message, core values, expert insights, and leadership perspectives. This section explains who we are, what we stand for, and the principles that guide our work.

## Services

The Services page provides a detailed explanation of all services offered by the company, including:

AI Product Development

Blockchain Development

Web and Mobile Development

AI Automation

White-Label AI Solutions

Custom GPT Development

The company delivers these services across multiple industries, including FinTech, Retail, Healthcare, Information Technology, and PropTech.

## Portfolio

The Portfolio section showcases the company’s solutions, completed projects, and technical expertise. It demonstrates real-world implementations and highlights the company’s ability to deliver scalable and effective solutions across various industries.

## Blog / Tech Talks

The Blog (Tech Talks) section contains articles and insights focused on emerging technologies and trends. Topics include AI in FinTech, the AI boom, and other AI-related discussions that reflect the company’s thought leadership and innovation.

## Careers

The Careers page highlights the company’s mission and workplace culture. It provides information on open opportunities, the application process, employee benefits, and what the company offers as part of its professional environment.

## Contact

The Contact page includes a structured contact form that allows users to submit inquiries, request services, or connect with the company for business and partnership opportunities.
